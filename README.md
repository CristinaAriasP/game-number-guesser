# 🎯 Game: Number Guesser

### Descripción
Juego sencillo de adivinanza de números. El objetivo es encontrar el número secreto en el menor número de intentos posible.

**Tecnologías usadas:**
* **JavaScript (ES6+)**: Lógica del juego y manipulación del DOM.
* **HTML5 & CSS3**: Estructura y diseño responsivo.
* **Git**: Gestión de versiones y flujo de trabajo.

### Description
Simple number guessing game. The goal is to find the secret number in as few attempts as possible.

**Technologies used:**
* **JavaScript (ES6+)**: Game logic and DOM manipulation.
* **HTML5 & CSS3**: Structure and responsive design.
* **Git**: Version control and workflow management.
